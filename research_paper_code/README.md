# Research Paper Code for Fertilizer Recommendation System

This folder contains enhanced implementations for the fertilizer recommendation system, incorporating advanced features for research publication.

## Files:
- `enhanced_traditional_ml.ipynb`: Traditional ML with ensemble methods, hyperparameter tuning, and sustainability metrics.
- `enhanced_bigdata_ml.ipynb`: Big data approach with deep learning, API integration, and distributed processing.
- `data_augmentation.py`: Script to generate synthetic data for better training.
- `sustainability_calculator.py`: Module to calculate environmental impact of fertilizer recommendations.
- `visualization_and_reporting.ipynb`: Comprehensive statistical analysis and report generation.
- `experiments_runner.py`: Automated script for running multiple experiments and comparisons.

## Setup:
1. Install requirements: `pip install -r requirements.txt` (for traditional) or `pip install -r bigdata_requirements.txt` (for big data).
2. Run notebooks in order for data augmentation, model training, evaluation, and reporting.
3. For APIs, get an OpenWeatherMap API key and set it in the code.

## Key Enhancements:
- Data augmentation for larger datasets
- Deep learning models (Neural Networks)
- Ensemble methods and hyperparameter tuning
- Real-time weather API integration
- Sustainability metrics (cost, emissions)
- Advanced visualizations and statistical reporting
- Comparative analysis of models
- Reproducibility with logging and random seeds
